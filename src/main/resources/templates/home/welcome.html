Bem vindo!

BootLTE é o resultado da integração do Spring Boot (utilizando o template engine Thymeleaf) com o tema AdminLTE. O propósito deste projeto é ser utilizado como ponto de partida para outros projetos utilizando o tema AdminLTE. Visite os links abaixo para mais informações.

  • Spring Boot 1.4.1
  • Thymeleaf 2.1.5
  • Thymeleaf Layout Dialect 1.4.0
  • AdminLTE 2.3.6
